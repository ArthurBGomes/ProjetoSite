Tópicos sobre HTML

HTML significa HyperText Markup Language — é a base da web e serve para estruturar textos, imagens e links. Não é uma linguagem de programação, mas sim de marcação, usada para dizer ao navegador como exibir o conteúdo.

Todo arquivo HTML tem as tags principais: <html>, <head> e <body>. O <head> guarda informações invisíveis, e o <body> contém tudo o que aparece na página.

As tags são os blocos do HTML. Exemplo: <p> cria parágrafos, <h1> títulos e <img> imagens. Juntas formam os elementos que estruturam a página.

Os atributos dão mais informações às tags, como src (imagem), href (link) e alt (descrição). Eles personalizam o comportamento de cada elemento.

Os textos usam <p> e os títulos variam de <h1> a <h6>, seguindo a hierarquia da informação. Isso ajuda na leitura e no SEO.

As listas podem ser ordenadas (<ol>) ou não ordenadas (<ul>). Cada item usa <li>. São usadas em menus, tópicos e instruções.

Os links usam a tag <a href="..."> para conectar páginas ou partes do site. São essenciais para a navegação na web.

As imagens são inseridas com <img> e usam src para o caminho e alt para o texto alternativo. Tornam o conteúdo mais visual e acessível.

Os formulários usam <form> e campos como <input>, <button> e <label> para coletar dados de usuários, como logins e mensagens.

A semântica dá significado ao código, usando tags como <header>, <main> e <footer>. Isso ajuda mecanismos de busca e acessibilidade.

HTML (HyperText Markup Language) é a linguagem usada para criar e estruturar páginas da web. Ele define o conteúdo que aparece em um site, como textos, imagens, links, tabelas e vídeos, usando tags que indicam a função de cada elemento. Todo documento HTML possui uma estrutura básica dividida em cabeçalho, onde ficam informações sobre a página, e corpo, que contém o conteúdo visível para o usuário.
As tags podem ter atributos, que adicionam detalhes como tamanho, cor, destino de links ou descrições de imagens. Com o avanço da linguagem, o HTML5 introduziu novas tags semânticas como <header> , <nav>, <main> e <footer>, que tornam o código mais organizado, acessível e fácil de entender por navegadores e mecanismos de busca. Além disso, o HTML permite criar formulários e tabelas para coleta e exibição de dados.
O HTML é a base da construção de sites e trabalha junto com outras tecnologias: o CSS, responsável pelo visual e design, e o JavaScript, que adiciona interatividade e dinamismo. Juntas, essas linguagens formam a estrutura principal da web moderna, possibilitando a criação de páginas funcionais, atraentes e bem organizadas.
