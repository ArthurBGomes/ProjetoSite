Tópicos sobre CSS

CSS (Cascading Style Sheets) é a linguagem usada para descrever a aparência e o layout de páginas da web. Ela trabalha em conjunto com o HTML, que define a estrutura e o conteúdo da página, enquanto o CSS controla cores, fontes, tamanhos, espaçamentos, bordas, alinhamentos, animações e muito mais. A palavra “Cascading” (em cascata) refere-se à forma como as regras de estilo são aplicadas: quando há várias declarações de estilo para o mesmo elemento, o navegador decide qual usar com base em uma hierarquia de prioridades (ordem de carregamento, especificidade e herança). O CSS permite separar o conteúdo (HTML) da apresentação visual (CSS), o que torna o código mais limpo, fácil de manter e reutilizável.

Inline (em linha): o estilo é aplicado diretamente em uma tag HTML através do atributo style.
Interno (no cabeçalho): o CSS é escrito dentro da tag "style" no "head" do documento HTML
Externo (arquivo .css): o CSS é colocado em um arquivo separado (por exemplo, style.css) e importado no HTML com a tag "link"

Texto: color, font-size, font-family, text-align, line-height, text-transform.
Cores e fundos: background-color, background-image, background-size, opacity.
Bordas e espaçamentos: border, border-radius, padding, margin.
Dimensões e posição: width, height, max-width, position, top, left, right, bottom.
Display e layout: display, flex, grid, float, clear.
Animações e transições: transition, animation, keyframes.

Seletores simples:
elemento: estiliza todas as tags com o nome especificado (p, h1, div).
.classe: estiliza elementos com uma classe específica (.menu, .botao).
#id: estiliza um elemento com um identificador único (#principal).

Todo elemento em CSS é tratado como uma caixa retangular composta por:
Content (conteúdo): onde o texto ou imagem aparece.
Padding: espaço entre o conteúdo e a borda.
Border: a borda que envolve o padding e o conteúdo.
Margin: espaço entre o elemento e os outros ao redor.
Entender o Box Model é essencial para alinhar e dimensionar corretamente os elementos na página.
É possível visualizar e alterar essas medidas no inspetor do navegador, ajustando o layout de forma precisa.
