Tópicos sobre HTML

avaScript é uma linguagem de programação usada para deixar os sites interativos e dinâmicos. Com ela, é possível criar animações, verificar formulários, mudar cores, textos e muito mais.
É utilizado principalmente em páginas da web, mas também pode ser usado em aplicativos, servidores e até jogos.
As variáveis servem para guardar informações, como números, textos ou valores verdadeiros/falsos. Existem três formas principais de criar variáveis: var, let e const.
Os principais tipos de dados em JavaScript são: Texto (string), Número (number), Verdadeiro ou falso (boolean), Nulo (null), Indefinido (undefined).
São usados para realizar operações. Podem ser de soma, subtração, comparação ou lógica (como “e”, “ou”, “não”).
Permitem que o programa tome decisões diferentes dependendo de uma condição. Exemplo: se algo for verdadeiro, faça uma ação; senão, faça outra.
São usados para repetir uma ação várias vezes. Os mais usados são: “for”, “while” e “do while”.
Funções são blocos de comandos criados para executar uma tarefa específica. Servem para evitar repetição de código e organizar o programa.
São listas que guardam vários valores dentro de uma única variável. Por exemplo, uma lista de frutas ou nomes.
Objetos guardam várias informações relacionadas a um mesmo tema. Por exemplo, nome, idade e cidade de uma pessoa.
É o que permite ao JavaScript interagir com o HTML da página. Com o DOM, o JavaScript pode alterar textos, imagens e cores em tempo real.
